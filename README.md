# Задание 2 — API-пайплайн: данные → LLM → результат

Скрипт читает новости из CSV-файла, отправляет каждую новость в LLM через OpenRouter API и сохраняет структурированный результат в JSON.

Задача — Саммаризация новостей: скрипт читит новости → LLM генерирует краткое содержание, категорию и ключевые факты → результат сохраняется в JSON.

## Запуск

1. Установить зависимости:
```bash
pip install openai python-dotenv

2. Создать файл .env с ключом:
OPENROUTER_API_KEY=Ваш_ключ

3. Подготовить файл news.csv с колонками id/title/text

4. Запустить скрипт:
py main.py news.csv

Результат сохраняется в news_results.json рядом с входным файлом.
